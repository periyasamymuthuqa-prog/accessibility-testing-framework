# Accessibility Testing Automation Framework


## Overview

Automation framework developed to validate web accessibility compliance using Playwright and Axe-core.

The framework performs automated accessibility scans based on WCAG 2.1 guidelines and generates accessibility violation reports.


## Technology Stack

- JavaScript
- Playwright
- Axe-core
- WCAG 2.1
- GitHub Actions


## Framework Features

- Automated accessibility scanning
- WCAG 2.1 rule validation
- Accessibility violation reporting
- Reusable accessibility utilities
- Screenshot capture
- CI/CD integration


## Accessibility Validations

- Missing alternative text
- Color contrast issues
- Keyboard accessibility issues
- Form label validation
- ARIA attribute validation
